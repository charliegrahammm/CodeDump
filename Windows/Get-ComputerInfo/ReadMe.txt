This is a small tool to show various bits of system information on the screen.
This is useful as it lets you see the local, public and VPN IP address of a computer - as well as other useful info like uptime.

A user can click on an item in the list and then click OK to copy it to the clipboard.

All the files in this folder are required.
- The Shortcut calls the CMD file, minimised.
- The ICO file is the icon for the shortcut
- The CMD file calls the PS1 file with executionpolicy bypassed.
